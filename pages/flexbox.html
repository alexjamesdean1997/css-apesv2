FLEXBOX

Full width image strips vertically stacked on screens below 960 px width, evenly distributed blocks on screens above 960 px using object-fit.

.strips > img {
  max-height: 200px;
  object-fit: cover;
  width: 100%;
}

@media screen and (min-width: 960px) {
  .strips {
    display: flex;
    justify-content: space-evenly;
    flex-wrap: wrap;
  }
  .strips > img {
    max-height: 200px;
    object-fit: none;
    width: auto;
    margin-top: 10px;
  }
}
